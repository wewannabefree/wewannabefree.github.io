Процесс создания модального окна из кнопки

  • Подключение стилей bootsrap:
    (link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css" integrity="sha384-BVYiiSIFeK1dGmJRAkycuHAHRg32OmUcww7on3RYdg4Va+PmSTsz/K68vbdEjh4u" crossorigin="anonymous")

  • Подключение библотеки jQuery:
    (script src="http://code.jquery.com/jquery-1.12.4.min.js" integrity="sha256-ZosEbRLbNQzLpnKIkEdrPv7lOy9C27hHQ+Xp8a4MxAQ=" crossorigin="anonymous">

  • Подключение js-файла Bootstrap:
    (script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js" integrity="sha384-Tc5IQib027qvyjSMfHjOMaLkfuWVxZxUPnCJA7l2mCWNIpG9mGCD8wGNIcPD7Txa" crossorigin="anonymous">

  • Создание кнопки
    1. Приписывание ей data-target="" - на какой id эта кнопка будет ссылаться
    2. Приписывание ей data-toggle="modal" - атрибут стилей Bootstrap, который используется для вызова модальных окон

  • Создание модального окна
    1. Взятие готового окна с сайта Bootstrap
    2. Приписывание ему id="" - на что будет ссылаться кнопка при нажатии
    3. Редактирование окна и заполнение его контентом:
      1. Заголовок окна (modal-title)
      2. Тело окна (modal-body)
      3. Нижняя часть окна (modal-footer)

    • Нижняя часть окна (modal-footer)

    Ссылка на скачанное модальное окно

    Это специальное модальное окно с сайта CodyHousе, скачанное отсюда:
    https://codyhouse.co/gem/svg-modal-window

    Процесс подключения такого окна к сайту:

  • Скачать архив с модальным окном отсюда:
    https://codyhouse.co/gem/svg-modal-window
  • Скопировать файлы в папку js:
    1. main.js
    2. snap.svg-min.js
  • Cкопировать картинки в папку img, если таковые имеются (в нашем случае это cd-icon-close.svg - кнопка закрытия)
  • Скопировать cd-modal (содержимое модального окна) с файла index.html из архива в наш документ
  • Скопировать стили из архивного файла style.css в наш main.less, начиная с абзаца "Modal Window"
  • Подключить скрипты двух вышесказанных js-файлов
